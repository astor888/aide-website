# 艾德知识产权 — 公司官网设计方案

## 一、项目概述

温州市艾德知识产权服务有限公司（简称：艾德）官方展示型网站。定位为高端制造业企业客户提供一站式企业服务展示平台。

## 二、核心诉求

| 维度 | 内容 |
|------|------|
| 目标客户 | 高端制造业企业（电气、汽摩配、五金制造、鞋服） |
| 市场定位 | 大企业 + 中小微企业 |
| 核心优势 | 流程快、通过率接近 100%、价格优势、AI 技术赋能 |
| 团队特点 | 90 后团队，踏实肯干 —— 用"真诚实干"差异化，不用传统权威背书 |
| 品牌印象 | 专业可靠 + 科技创新 + 高端大气 |

## 三、视觉设计方向

**A+B 融合方案：商务稳重 × 科技前瞻**

| 设计元素 | 方案 |
|----------|------|
| 主色 | 深炭灰/深蓝灰底色 |
| 文字色 | 冷白色为主 |
| 点睛色 | 香槟金 + 淡蓝渐变 |
| 字体策略 | 标题粗重无衬线体 / 正文纤细干净 / 关键数据用香槟金放大 |
| 设计语言 | 商务稳定大框架 + 科技细节（视差滚动、卡片悬浮动效、几何装饰） |
| 整体感受 | 像高端咨询公司一样可靠，又有科技公司的创新锐气 |

## 四、页面结构

### 导航
```
首页 | 关于我们 | 产品服务 | 客户案例 | 新闻动态 | 联系我们
```
预留：内部培训入口（已有系统，留接口）

### 首页 — 第一屏
**核心价值主张 + 服务入口导航 + 视觉冲击（方案 B+C 融合）**

- 大面积品牌视觉冲击（深色底 + 几何线条/粒子动效）
- 一句话核心主张：「制造业企业服务专家，项目申报通过率近 100%」
- 下方服务入口快速导航卡片（知识产权 / 项目申报 / AI 增效 / 企业服务）
- 关键数据滚动展示（服务行业数、通过率、客户数量）

### 关于我们 —「理念与承诺」
- 不放传统专家履历，走真诚实干路线
- 服务承诺 + 真实数据 + 工作场景
- 响应时间、服务客户数、申报成功率等硬数据
- 团队工作真实抓拍或办公环境照片
- 流程可视化 —— 客户能看到完整服务链路
- 团队文化：90 后，踏实肯干

### 产品服务
四大板块：

1. **知识产权服务**（专利、商标、版权等）
2. **政府政策项目申报**
3. **AI 应用增效服务**
4. **其他企业服务**

每板块包含：服务介绍 → 服务流程 → 为什么选我们 → 相关案例

### 客户案例
- 覆盖行业：电气、汽摩配、五金制造、鞋服
- 案例内容：客户行业背景 + 服务内容 + 成果数据
- 可先使用匿名/脱敏案例，数据真实即可
- 通过率接近 100% 作为贯穿全站的核心数字

### 新闻动态
- 行业政策解读
- 公司动态
- 知识科普（专利/申报相关知识）

### 联系我们
- 公司地址、电话、邮箱
- 在线表单
- 地图嵌入

### 内部培训入口（预留）
- 导航栏留入口
- 点击跳转至已有培训系统

## 五、技术实现

### 架构：前后端分离 Headless CMS

```
┌─────────────────┐      API       ┌──────────────────┐
│   前端展示站     │  ←────────→   │   Strapi 后台     │
│  (独立设计)      │   内容数据     │  (内容管理)       │
└─────────────────┘                └──────────────────┘
  用户看到的网站                     你登录修改内容的地方
```

- **前端**：独立设计实现，通过 API 拉取后台内容，可随时换设计
- **后台**：Strapi（开源免费，中文界面），你可登录后台随时修改页面内容
- **响应式**：适配 PC 和移动端
- **动画**：CSS 优先，性能优化
- **SEO**：静态生成，搜索引擎友好
- 你只需在后台像编辑 Word 一样修改内容，前端自动更新
